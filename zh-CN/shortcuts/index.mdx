# 快捷键

Reqable支持大量的快捷操作，按照功能分为下面几个类别：
- [Tabs操作](#tabs)
- [侧边栏](#sidebar)
- [调试控制](#capture-exec)
- [调试列表](#capture-list)
- [API请求](#rest)
- [编辑器](#editor)
- [规则列表](#rule-list)
- [其他](#others)

:::info
本文档列出的快捷键可能不完整，具体请留意应用内的快捷键提示。
:::

### Tabs操作 {#tabs}
| 说明 | 按键 |
| ---- | ---- |
| 新建API         | <Shortcut>Control + T</Shortcut> |
| 关闭当前Tab      | <Shortcut>Control + W</Shortcut> |
| 关闭全部Tab      | <Shortcut>Shift + Control + W</Shortcut> |
| 按照索引切换Tab   | <Shortcut>Control + 1/2/3/4/5/6/7/8/9</Shortcut> |
| 切换到右侧Tab    | <Shortcut>Control + 0</Shortcut> |
| 向左切换Tab    | <Shortcut>Control + [</Shortcut> |
| 向右切换Tab    | <Shortcut>Control + ]</Shortcut> |
| 切换到上一次的Tab    | <Shortcut>Control + -</Shortcut> |
