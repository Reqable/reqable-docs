# 主题

Reqable支持丰富的主题设置，可以在 设置 -> 主题 中配置您喜欢的主题方案：

![](arts/settings.png)

### 外观

Reqable支持三种模式：亮色、暗色和跟随系统（即使用系统当前的模式）。

![亮色效果](arts/appearence.png)

### 代码配色

Reqable支持十六种代码配色：

- Atom One
- Classic
- Edge
- Equilibrium
- Github
- Google
- Harmonic16
- Humanoid
- Paper Color
- Silk
- Stackoverflow
- Summer Fruit
- Tokyo Night
- Unikitty
- VS Code

默认采用Atom One配色，如果您有其他的配色需求，请在此[issue](https://github.com/reqable/reqable-app/issues/17)下留言。

### 语言

Reqable目前仅支持下面两种语言：
- 简体中文
- 英语
