# Shortcuts

Reqable supports shortcut keys, which are divided into the following categories:
- [Tabs](#tabs)
- [Sidebar](#sidebar)
- [Capture](#capture-exec)
- [Traffic List](#capture-list)
- [API Testing](#rest)
- [Editor](#editor)
- [Rule List](#rule-list)
- [Others](#other)

:::info
The shortcut keys listed in this document may not be complete, please pay attention to the shortcut key tips in the app.
:::

### Tabs {#tabs}
| Description | Keys |
| ---- | ---- |
| New API         | <Shortcut>Control + T</Shortcut> |
| Close tab      | <Shortcut>Control + W</Shortcut> |
| Close all tabs      | <Shortcut>Shift + Control + W</Shortcut> |
| Switch tab with index   | <Shortcut>Control + 1/2/3/4/5/6/7/8/9</Shortcut> |
| Switch to the last tab    | <Shortcut>Control + 0</Shortcut> |
| Switch tab to the left    | <Shortcut>Control + [</Shortcut> |
| Switch tab to the right    | <Shortcut>Control + ]</Shortcut> |
| Switch to the previous tab    | <Shortcut>Control + -</Shortcut> |
